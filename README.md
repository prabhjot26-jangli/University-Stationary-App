# University-Stationary-App
This helps students to know their printing status for the files given for printing at the university shop , rather than visiting everything and asking if their documents have been printed or not, it uses real time database or show the live status of their printing process.
Students can easily login and access their profiles via the student dasboard, which will help them save their plenty of time
